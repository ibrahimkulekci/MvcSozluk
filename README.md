# MvcSozluk - İnteraktif Sözlük Yazılımı V1

## Kullanılan Teknoloji ve Yapılar

1. ASP.NET MVC5
2. Entity Framework (CodeFirst)
3. N-Tier Architecture (N Katmanlı Mimari)
4. Sql Linq Query 
5. Bootstrap (Tema: AdminLTE-3.0.4)
